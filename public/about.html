GitLab Pipeline Status v2

Estado en tiempo real de pipelines de GitLab. Visualiza branches y sus pipelines con actualizaciones automaticas via Server-Sent Events (SSE).

Stack

  • Runtime: Bun
  • Backend: Hono (API JSON + SSE)
  • Frontend: Preact + htm (ESM, sin build step)
  • Tiempo real: Server-Sent Events (SSE)

Funcionalidades

  • Vista por servidor y proyecto
  • Detalle de branches y estado del pipeline
  • Actualizaciones en tiempo real via SSE
  • Suscripcion selectiva por proyecto
  • Estado de tokens (salud y fallback)
  • Modo oscuro automatico
  • Responsive (movil y escritorio)
